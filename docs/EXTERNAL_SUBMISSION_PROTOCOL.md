# External Scientific Submission Boundary

**Status:** Blocked.

No candidate, report, data package, alert, telescope request, journal or
preprint submission, public disclosure, or authority-facing communication is
authorized by this repository. No current candidate has cleared the required
review chain, and the standalone escalation gate has no admissible calibrated
SNR threshold and fails closed.

## Scope

This document is a stop boundary, not an executable submission procedure. A
local candidate packet, pathway value, score, anomaly rank, successful test
run, or operator review never authorizes external action.

The following are external actions and remain blocked:

- contacting a telescope facility, survey team, SETI organization, journal,
  preprint server, public forum, press outlet, or government/standards body;
- posting candidate evidence or source data as a signal claim;
- requesting follow-up observations from an external party;
- describing an unresolved result as a detection, discovery, confirmation, or
  independently validated candidate.

Routine publication of source code and clearly scoped negative methodology is
not candidate submission, but it must not disclose restricted data or imply
scientific confirmation.

## Required Review Chain

Even a future strong local result cannot advance externally unless all of the
following are documented for that specific result:

1. The automated multi-modal pipeline has produced a provenance-complete local
   candidate packet from real observations.
2. Track A known-explanation checks and Track B evidence gates have passed
   without fabricated, inferred, synthetic, or project-created ground truth.
3. Radio-frequency interference, satellite/transmitter, cadence, drift,
   cross-target, instrument-artifact, catalog, and data-quality explanations
   applicable to the modality have been tested and preserved in the packet.
4. Reproduction uses checksum-matching source artifacts and a frozen pipeline
   version, with limitations and abstentions recorded.
5. A purpose-built adversarial review agent has attempted to refute the result
   using every applicable mundane, natural, and instrumental explanation.
6. The result has survived credentialed third-party expert review through an
   appropriate venue selected for the signal type.
7. The user has explicitly approved the particular external action after
   reviewing its destination, contents, claims, risks, and data rights.

Passing local gates does not satisfy steps 5-7 automatically. The agent must
stop for the user's authorization before any external scientific or official
submission, contact, or disclosure.

## Current Blockers

- There are zero independently escalation-ready candidates.
- No SNR escalation threshold is calibrated from admissible evidence.
- Learned/anomaly outputs are uncalibrated local ranking diagnostics.
- No candidate-specific adversarial review has been required or completed.
- No credentialed third-party expert has confirmed a candidate.
- No external action has been approved.

The correct current activity is deterministic analysis, false-positive
rejection, local evidence preservation, and honest negative-result reporting.
No person should be asked to create missing labels or calibration truth.
